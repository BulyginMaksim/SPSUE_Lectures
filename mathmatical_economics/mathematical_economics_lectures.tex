\section{Введение}
\subsection{Проблемы, цели и задачи}
\term{Математическая экономика} -- математическая дисциплина, предметом которой являются модели экономических объектов, процессов и систем,
а также методы их исследования.

\term{Математическая экономика} -- направление в теоретической экономике, использующее математические методы и модели
для описания экономических процессов.

Основные понятия:
\begin{itemize}
	\item \term{Модель} (от лат. modulus -- мера, норма, образец, аналог) -- образ (в т.ч. условный или мысленный -- изображение,
	описание, схема, чертеж, график, план, карта и т.п.) или прообраз (образец) какого-либо объекта или системы объектов, используемый
	при определенных условиях в качестве их <<заместителя>> или <<представителя>>, объект-заместитель изучаемой системы.
	
	\item \term{Математическая модель} -- описание объекта исследования на языке математики.
	
	\item \term{Математическое моделирование} -- средство исследования сложных систем и объектов различной природы
	на основе их математических моделей. Для экономических систем и процессов -- это основной инструмент изучения и анализа.
\end{itemize}

Информационный взрыв и <<первобытное>> мышление: обратная сторона информационной революции -- постепенное смещение к мифологизированному
первобытному мышлению, переключение с критического осмысления события или явления на сопереживательно-эмоциональное. Сужение сознания человека: оно становится фрагментированным, мозаичным, быстро 
теряется целостность восприятия.
Один из главных вызовов -- способность качественно осмысливать информацию, синтезируя из нередко противоречивой мозаичной картинки целостное представление.

Требования к математическим моделям:
\begin{itemize}
	\item Адекватность;
	\item Конечность;
	\item Полнота (информативность);
	\item Упрощенность;
	\item Гибкость;
	\item Приемлемая трудоемкость разработки.
\end{itemize}

Построение математических моделей сложных систем -- слабо формализуемый процесс. Можно выделить общую схему действий, этапы, но привести универсальный алгоритм, позволяющий построить математическую модель произвольной
сложной системы невозможно.


Этапы построения математической модели:
\begin{enumerate}
	\item Определение цели процесса моделирования;
	\item Изучение предметной обалсти, выявление причинно-следственных связей, построение концептуальной модели;
	\item Переход от концептуальной модели к формализованному описанию;
	\item Проверка адекватности модели;
	\item Корректировка модели;
	\item Применение модели, проведение исследований и практическое использование;
	\item Уточненение и улучшение модели.
\end{enumerate}

Подходы к моделированию:
\begin{itemize}
	\item Аналитический;
	\item Экспериментальный;
	\item Аналитико-экспериментальный.
\end{itemize}

\subsection{Производственные функции}

На входе эмпирические данные (выборка) $\overline{X} = (x_1, \cdots , x_n)$, на выходе конечные, прогнозные данные $\overline{Y} = (y_1,\cdots,y_m)$.
\begin{figure}[h]
	\centering
	\includegraphics[width=0.4\linewidth]{pics/screenshot001}
	\label{fig:screenshot001}
\end{figure}

\term{Производственная функция $\overline{Y} = F(\overline{X})$} -- функция, выражающая зависимость между затратами ресурсов и объемами выпуска, 
где $\overline{X}$ -- вектор используемых ресурсов, $\overline{Y}$ -- вектор объемов выпуска продукции каждого вида.

Предполагается, что ПФ $F(x_1,\cdots,x_n)$ является достаточно гладкой -- по крайней мере, дважды непрерывно дифференцируема. Тогда, из предположения вытекают
следующие свойства неоклассических ПФ:

\begin{enumerate}
	\item $F(x_1,\cdots,x_n) = 0$, если $x_i = 0$ хотя бы для одного $i = 1,2,\cdots,n$ (при отсутствии хотя бы одного ресурса производство невозможно).
	\item $F(x_1,\cdots,x_n)$ -- возрастающая функция по каждому аргументу: 
	$$\frac{\partial F}{\partial x_i}>0 \;\; \forall i: i=1,\cdots,n$$
	\item Выпуск по каждому аргументу неограничен.
	\item Предельная производительность факторов убывает: 
	$$\frac{\partial^2 F}{\partial x^2_i}<0 \;\; \forall i: i=1,\cdots,n$$
	\item Линейная однородность или отадча от масштаба:
	$$F(\lambda x_1,\cdots,\lambda x_n) = \lambda F(x_1,\cdots,x_n)$$ \\
	В общем случае (линейное при $\alpha = 1$):
	$$F(\lambda x_1,\cdots,\lambda x_n) = \lambda^\alpha F(x_1,\cdots,x_n)$$
\end{enumerate}

Макроэкономические ПФ: целевой показатель -- $Y$ (валовый внутренний продукт), входы модели $K$ (основные производственные фонды), $L$ (число занятых).

\begin{figure}[h]
	\centering
	\includegraphics[width=0.4\linewidth]{pics/screenshot002}
	\label{fig:screenshot002}
\end{figure}

Производственная функция Кобба-Дугласа:
$$Y = F(K,L) = A K^\alpha L^\beta$$
Где  $A>0, \; 0 <\alpha < 1,\; 0 <\beta <1$  обладает свойствами 1-4.

Ч.Кобб и П.Дуглас, используя данные по обрабатывающей промышленности США двадцатых годов прошлого века, получили оценки для $\alpha = 0.27, \beta \geqslant 0.73$. 

Оценим параметры $A,\alpha,\beta$ производственной функции Кобба-Дугласа $Y = A K^\alpha L^\beta$:
\begin{itemize}
	\item Для оценки параметров функции будем использовать метод наименьших квадратов.
	\item Потребуются исторические данные $(K_i, L_i, Y_i)^{i=M}_{i=1}$.
\end{itemize}

Для использования МНК линеаризуем ПФ Кобба-Дугласа, тогда линеаризованная форма ПФ Кобба-Дугласа имеет вид:
$$ Y = A K^\alpha L^\beta$$
$$ \ln Y = \ln \left(A K^\alpha L^\beta\right)$$
$$\ln Y = \ln A + \alpha \ln K + \beta \ln L$$
$$S(A, \alpha, \beta) = \sum_{i=1}^{M}  (\ln A + \alpha \ln K_i + \beta \ln L_i - \ln Y_i)^2 =  \sum_{i=1}^{M} \varepsilon^2 \rightarrow min$$


	$$\begin{cases}
		\displaystyle \dfrac{\partial S}{\partial \ln A} = 0 \\[8pt]
		
		 \displaystyle\dfrac{\partial S}{\partial \alpha} = 0\\[7pt]
		
		 \displaystyle\dfrac{\partial S}{\partial \beta} = 0
	\end{cases}$$


\begin{equation*}
	\begin{cases}
		\displaystyle 2 \sum_{i=1}^{M}  (\ln A + \alpha \ln K_i + \beta \ln L_i - \ln Y_i) \cdot 1 = 0\\
		\displaystyle2 \sum_{i=1}^{M}  (\ln A + \alpha \ln K_i + \beta \ln L_i - \ln Y_i) \cdot \ln K_i = 0\\
		\displaystyle 2 \sum_{i=1}^{M}  (\ln A + \alpha \ln K_i + \beta \ln L_i - \ln Y_i) \cdot \ln L_i = 0\\
	\end{cases}
\end{equation*}


\begin{equation*}
	\begin{cases}
		\displaystyle \ln A \cdot M + \alpha \cdot\sum_{i=1}^{M}\ln K_i + \beta \cdot\sum_{i=1}^{M}\ln L_i = \sum_{i=1}^{M}\ln Y_i\\
		\displaystyle\ln A \cdot\sum_{i=1}^{M}\ln K_i + \alpha\cdot \sum_{i=1}^{M}\ln^2 K_i + \beta \cdot\sum_{i=1}^{M}\ln K_i \ln L_i = \sum_{i=1}^{M}\ln Y_i \ln K_i\\
		\displaystyle\ln A \cdot\sum_{i=1}^{M}\ln L_i + \alpha\cdot \sum_{i=1}^{M}\ln K_i \ln L_i + \beta \cdot\sum_{i=1}^{M} \ln^2 L_i = \sum_{i=1}^{M}\ln Y_i \ln L_i\\
	\end{cases}
\end{equation*}

Параметрическая идентификация: $B \cdot \overline{C} = \overline{Z}$

$$\overline{C} = \begin{pmatrix}
	\ln A\\
	\alpha\\
	\beta
\end{pmatrix},  
\; \; \overline{Z} = \begin{pmatrix}
	\sum\limits_{i=1}^{M} \ln Y_i\\
	\sum\limits_{i=1}^{M} \ln Y_i \ln K_i\\
	\sum\limits_{i=1}^{M} \ln Y_i \ln L_i
\end{pmatrix}$$

$$B = \begin{pmatrix}
	M & \sum\limits_{i=1}^{M}\ln K_i & \sum\limits_{i=1}^{M}\ln L_i\\
	\sum\limits_{i=1}^{M}\ln K_i & \sum\limits_{i=1}^{M}\ln^2 K_i & \sum\limits_{i=1}^{M}\ln K_i \ln L_i\\
	\sum\limits_{i=1}^{M}\ln L_i & \sum\limits_{i=1}^{M}\ln K_i \ln L_i & \sum\limits_{i=1}^{M}\ln^2 L_i\\
\end{pmatrix}$$

Тогда матрица оценок параметров будет иметь вид $\overline{C} = B^{-1} \cdot \overline{Z}$.

Часто рассматривают $\beta = 1-\alpha$, тогда ПФ Кобба-Дугласа имеет вид:
$$Y = A K^{\alpha} L^{1-\alpha}, \; 0 < \alpha < 1$$

\begin{itemize}
	\item Для оценки параметров функции будем использовать метод наименьших квадратов.
	\item Потребуются исторические данные $(K_i, L_i, Y_i)^{i=M}_{i=1}$.
\end{itemize}


Линеаризованная форма ПФ Кобба-Дугласа с $\beta = 1 - \alpha$: 

$$\ln Y = \ln A + \alpha \ln K + (1-\alpha) \ln L$$

$$\ln \left(\frac{Y}{L}\right) = \ln A + \alpha \ln \left(\frac{K}{L}\right) \Rightarrow z = a + \alpha \cdot u$$

Почему $\alpha + \beta = 1$? Экономический смысл: рассматривая удельные показатели $\frac{Y}{L}$ -- производительность и $\frac{Y}{K}$ -- капиталовооруженность. Идея состоит в том, чтобы
уйти от учета количества занятых ($L$), если мы хотим охарактеризовать общее свойство экономики: экономики будут различаться не числом занятых, а от соотношения производительности и 
фондовооруженности.



\term{Множество безразличия производителя} (изокванты) -- множество наборов производственных факторов, при использовании которых уровень производства не изменяется.

Для двух факторов -- линии равного выпуска продукции на координатной плоскости (в осях $K,L$) -- линии равного уровня выпуска продукции.

\textit{\textbf{Анализ производственных функций}}. 

Средние производительности факторов:
$$A_{FL} = \dfrac{F(K,L)}{L} \; \; \; \; A_{FK} = \dfrac{F(K,L)}{K}$$

Предельные производительности факторов (величина, на которую изменится выпуск, если соответствующий фактор изменится на единицу при фиксированных значениях остальных факторов):
$$M_{FL} = \dfrac{\partial F(K,L)}{\partial L}  \; \; \; \; M_{FK} = \dfrac{\partial F(K,L)}{\partial K}$$

Для ПФ функции Кобба-Дугласа:

$$M_{FL} = \beta \dfrac{Y}{L} \; \; \; \; M_{FK} = \alpha \dfrac{Y}{K}$$

Эластичность выпуска по факторам (на сколько процентов изменится объем выпуска при изменении соответствующего фактора на один процент): 

$$ \varepsilon_{K} = \dfrac{\partial \ln F(K,L)}{\partial \ln K} = \dfrac{M_{FK}}{A_{FK}} = \alpha $$
$$ \varepsilon_{L} = \dfrac{\partial \ln F(K,L)}{\partial \ln L} = \dfrac{M_{FL}}{A_{FL}} = \beta $$

Замещение факторов: на изокванте значения ПФ постоянны, поэтому:

$$F(K, L) = const \Rightarrow dF = 0$$
$$dF = \dfrac{\partial F}{\partial K} dK + \dfrac{\partial F}{\partial L} dL = 0$$

$$MRS_K = -\dfrac{dK}{dL}\bigg|_{F=const} = \dfrac{\partial F}{\partial L} \big/ \dfrac{\partial F}{\partial K} = \dfrac{\beta K}{\alpha L}$$
$$MRS_L = -\dfrac{dL}{dK}\bigg|_{F=const} = \dfrac{\partial F}{\partial K} \big/ \dfrac{\partial F}{\partial L} = \dfrac{\alpha L}{\beta K}$$
Эластичность замещения факторов:

$$\delta_{K,L} = \dfrac{d\left(K/L\right)}{K/L}\big/\dfrac{d(MRS)}{MRS}$$

\textit{\textbf{ПФ с постоянной эластичностью замещения факторов (ПФ CES).}}

$$Y = A(bK^{-\rho}+(1-b)L^{-\rho})^{-\frac{\gamma}{\rho}}$$


Где $\sigma = \dfrac{1}{1+\rho} \text{эластичность замещения факторов}, \; 0<\gamma\leqslant 1, \; 
\rho>-1, \;  0<b\leqslant1$.

Если $\gamma = 1$ и $\rho \rightarrow 0 \Leftrightarrow\sigma \rightarrow 1$, то получаем ПФ Кобба-Дугласа.

Если $\rho \rightarrow \infty \Leftrightarrow \sigma \rightarrow 0 $, то получаем ПФ Леонтьева (т.к. абсолютно негибкая ПФ, факторы незаменяемые) $Y = A\min \left(\dfrac{K}{K_0},\dfrac{L}{L_0}\right)$.

Если $\rho \rightarrow -1 \Leftrightarrow \sigma \rightarrow \infty$, то получаем линейную ПФ $Y = A(bK+(1-b)L)$.

\subsection{Модель Леонтьева (модель межотраслевого баланса)}

Исходными данными модели являются статистические таблицы, в которых содержатся  данные об объемах конечной продукции (сколько какого продукта
выпущено) и валовые отраслевые выпуски.

Например, фиксируем данные о производстве горючего. Для того чтобы наладить выпуск, нужна доставка и расход продукта. Тот продукт, который выпускается в качестве конечного фиксируется отдельно, а промежуточные затраты, необходимые для производства конечного продукта, тоже фиксируются.

Данные фиксируются в стоимостном и натуральном выражении, но мы будем предполагать, что данные представлены в стоимостном выражении.
В строчках таблицы -- межотраслевые объемы (сколько выпущено), в столбиках -- сколько потрачено в соответствующей отрасли.

Предпосылки: в экономике $n$ отраслей; каждая отрасль производит единственный вид продуцкии и потребляет продукцию других отраслей (которую она потребляет в процессе производства единственного вида продукции); разные отрасли производят разные виды продукции.

Обозначения:
\begin{itemize}
	\item $x_{ij}$ -- объем продукции (в стоимостном выражении), произведенной в отрасли $i$ и потребляемой отраслью $j$;
	\item $X_i$ -- валовый продукт отрасли $i$;
	\item $Y_i = F_i$ -- конечный продукт отрасли $i$.
\end{itemize}

Баланс -- все, что выпустили, то и потребили (потребность полностью удовлетворена).

Межотраслевой баланс (показывает, на что израсходован выпуск $i$-ой отрасли):
$$X_i = \sum_{j=1}^{n} x_{ij} + F_i \;\; \; \text{где } i,j = 1, \cdots, n$$

\term{Коэффицинт прямых затрат} -- затраты продукции $i$-ой отрасли, которые необходимы для производства единицы продукции $j$-ой отрасли.

$$a_{ij} = \dfrac{x_{ij}}{X_j} \Rightarrow
X_i = \sum_{j=1}^{n} a_{ij}X_j + F_i \;\; \; \text{где } i,j = 1, \cdots, n$$

Тогда возможно представление в матричной форме:
$$X = A\cdot X + F, \; \; \; X, F \ge 0$$

Как использовать межотраслевой баланс? Если знаем валовые выпуски, можем определить конечное потребление; если знаем конечное потребление, можем сделать прогноз на валовый выпуск.
$$F =  (E  - A) \cdot X$$
$$X = (E - A)^{-1} \cdot F$$

Матрицы $A$, которые гарантируют $\forall F \Rightarrow \forall X>0$, называются \term{продуктивными}.

Условия продуктивности матрицы $A$:
\begin{enumerate}
	\item Если хотя бы для одного положительного вектора $Y$ уравнение межотраслевого баланса $X = A\cdot X+Y$ имеет неотрицательное решение $X$, то матрица $A$ продуктивна.
	\item Для продуктивности матрицы $A$ необходимо и достаточно существования и неотрицательности матрицы $(E-A)^{-1}$.
	\item Неотрицательная матрица $A$ продуктивна тогда и только тогда, когда матрица $(E-A)$ имеет $n$ положительных последовательных главных миноров.
	\item Неотрицательная матрица $A$ продуктивна тогда и только тогда, когдая сходится матричный ряд $E + A + A^2 + \cdots +
	A^k + \cdots$.
\end{enumerate}

\term{Обратная матрица Леонтьева} $B = (E-A)^{-1}$ -- это матрица полных затрат. Элементы этой матрицы $b_{ij}$ -- это количество продукции отрасли $i$, используемое для производства единицы конечного продукта отрасли $j$.

\subsection{Модель Солоу-Свана}

\term{Модель Солоу-Свана} -- модель экзогенного экономического роста.

Предположения и обозначения:
\begin{itemize}
	\item Экономика замкнута;
	\item Экономика односекторна: $Y$ -- валовый внутренний продукт;
	\item $K$ -- капитал;
	\item $I$ -- инвестиции;
	\item $C$ -- конечно потребление;
	\item $L$ -- трудовые ресурсы;
	\item Имеет место баланс: $Y = C + I$;
	\item Зависимость валового внутреннего продукта от ресурсов
	выражается производственной функцией Кобба-Дугласа $Y = A K^{\alpha} L^\beta$.
\end{itemize}

Ресурсы:
\begin{itemize}
	\item Труд $L$ (численность рабочих, занятых в производстве) вводится в систему экзогенно и растет с постоянным темпом ($\gamma$ -- темп прироста трудовых ресурсов);
	$$L(t + \Delta t) = L(t)  +\gamma \cdot L(t) \cdot \Delta t$$
	$$\Delta L = \gamma \cdot L \cdot \Delta t $$
	$$\dfrac{dL}{dt} = \gamma L$$
	$$L = L_0 e^{\gamma t}$$
	\item Динамика фондов $K$:
	за год фонды уменьшаются за счет износа на величину
	$\mu K$, где $\mu$ -- доля фондов, которые выбывают за год 
	(норма амортизации $0 < \mu < 1$), а увеличение фондов в течение года равно годовому объему инвестиций (с нулевым лагом)
	$$\Delta K = -\mu \cdot K \cdot \Delta t + I \cdot \Delta t$$
	$$\dfrac{dK}{dt} = -\mu \cdot K + I $$
	\item Инвестируется фиксированная доля ВВП ($s$ -- склонность к сбережению, $0 \le s \le 1$)
	$$ I = sY$$
\end{itemize}

\subsection{Модель Самуэльсона-Хикса}

\textit{\textbf{Модель Кейнса.}}

Предположения модели:
\begin{itemize}
	\item Экономика замкнута;
	\item Предложение эластично;
	\item Цены и процентная ставка фиксированы;
	\item Потребление в текущем периоде определеяется доходом в предществующем периоде.
\end{itemize}

$Y(t)$ -- валовый выпуск экономики в момент времени $t$, $C(t)$ -- спрос на потребительские товары, $I(t)$ -- спрос на инвестиции, предполагаем, что $\Delta t = 1$,
спрос $C(t)$ зависит от $Y(t)$ линейно, а инвестиции постоянны $I(t) = I$.

Считаем, что $Y(t+1) = C(Y(t)) + I(t)$.

Пусть спрос зависит от выпуска линейно, инвестиции фиксированы:
$$Y(t+1) = C_a + c\cdot Y(t) + I, \;\; 0 \le c \le 1$$

Сначала внесем динамику и рассмотрим приращение $Y$ во времени:
$$Y(t+1) = C_a + c\cdot Y(t) + I$$
$$Y(t+1) - Y(t) = C_a + c\cdot Y(t) - Y(t) + I$$

Если $\Delta t \not = 1$:
$$Y(t+1) - Y(t) = (C_a + c\cdot Y(t) - Y(t) + I) \cdot \Delta t$$
$$\dfrac{dY}{dt} = C_a + c\cdot Y(t) - Y(t) + I$$
$$\dfrac{1}{1-c} \cdot \dfrac{dY}{dt} = \dfrac{C_a + I}{1-c} - Y(t)$$

Если $y(t) = const$:
$$Y(t) = \dfrac{C_a + I}{1-c}$$

\textit{\textbf{Модель Самуэльсона-Хикса.}}

Предположения:
\begin{itemize}
	\item Пусть динамика инвестиций определяется динамикой ВВП:
	$$I(t) = r\cdot(Y(t) - Y(t-1)) + I_a$$
	\item $r$ -- коэффициент акселерации, значение которого равно величине изменения инвестиций
	при изменении ВВП на одну единицу
	\item $Y(t+1) = C_a + c\cdot Y(t) + r \cdot (Y(t) - Y(t-1)) + I_a$
	\item $Y(t+1) = c\cdot Y(t) + r \cdot (Y(t) - Y(t-1)) + A$
\end{itemize}

Пусть $\Delta t \not = 1$:
$$Y(t+1) = c\cdot Y(t) + r \cdot (Y(t) - Y(t-1)) + A$$
$$Y(t+1) - 2Y(t) + Y(t-1) =C_a + I_a - (1-c) \cdot Y(t) - (1-r)(Y(t) - Y(t-1))$$
$$Y(t+1) - 2Y(t) + Y(t-1) =\left(C_a + I_a - (1-c) \cdot Y(t) - (1-r)(Y(t) - Y(t-1))\right) \Delta t$$
$$Y(t+\Delta t) - 2Y(t) + Y(t-\Delta t) =\left(C_a + I_a - (1-c) \cdot Y(t)) (\Delta t)^2  - (1-r)(Y(t) - Y(t-\Delta t))\right) \Delta t$$
$$\dfrac{1}{\Delta t} \left(\dfrac{Y(t+\Delta t) - y(t)}{\Delta t} - \dfrac{Y(t) - Y(t-\Delta t)}{\Delta t}\right) = C_a + I_a - (1-c) \cdot Y(t) - (1-r)\dfrac{Y(t) - Y(t-1)}{\Delta t}$$
$$\dfrac{d^2Y}{dt^2} = C_a + I_a - (1-c) \cdot Y(t) - (1-r)\dfrac{dY}{dt}$$

Состояние равновесия (положим $Y(t) = C_0$):
$$C_a + I_a - (1-c) \cdot C_0 = 0$$
$$C_0 = \dfrac{C_a + I_a}{1-c}$$
$$Y(t) = \dfrac{C_a + I_a}{1-c}$$

Рассмотрим:
$$\dfrac{d^2Y}{dt^2} = C_a + I_a - (1-c) \cdot Y(t) - (1-r)\dfrac{dY}{dt}$$

Характеристическое уравнение:
$$\lambda^2 + (1-r)\cdot\lambda + (1-c) = 0$$

Исходя из корней характеристического уравнения делаем вывод о динамике $Y(t)$.

Количественный анализ:

$$\dfrac{1}{1-c} \cdot \dfrac{d^2Y}{dt^2}  + \dfrac{1-r}{1-c} \cdot \dfrac{dY}{dt} + Y = \dfrac{A}{1-c}$$

Замена $\dfrac{dY}{dt} = u$:

$$\dfrac{du}{dt} = \dfrac{d^2Y}{dt^2}$$
$$\dfrac{d^2Y}{dt^2} = \left(-\dfrac{1-r}{1-c} \cdot \dfrac{dY}{dt} - Y + \dfrac{A}{1-c} \right)(1-c)$$

\begin{equation*}
	\begin{cases}
		\displaystyle \dfrac{du}{dt} = \left(-\dfrac{1-r}{1-c} \cdot \dfrac{dY}{dt} - Y + \dfrac{A}{1-c} \right)(1-c)\\[8pt]
		\displaystyle \dfrac{dY}{dt} = u\\
	\end{cases}
\end{equation*}

Предположения:

\term{Функция полезности} $U(\overline{X}$: $U(\overline{X}_1) > U(\overline{X}_2)$, если 
$\overline{X}_1 \succ \overline{X}_2$.

Свойства функции полезности $U(\overline{X}$:
\begin{enumerate}
	\item Функция растет по каждому аргументу;
	\item Предельная полезность убывает.
\end{enumerate}

Задача потребительского выбора:
$$U(x_1, \cdots, x_n )\rightarrow \max$$
$$\sum_{i=1}^{n} x_i p_i \le I$$
$$x_i \ge 0$$


Но на самом деле:
$$U(x_1, \cdots, x_n )\rightarrow \max$$
$$\sum_{i=1}^{n} x_i p_i = I$$
$$x_i \ge 0$$

Решая такую оптимизационную задачу (для отдельно взятого потребителя) мы получаем
\term{функцию спроса}: $x_i = x_i(p_1, \cdots, p_n, I)$. Функции спроса, как и функции полезностей, у разных потребитилей будут разными.

Метод множителей Лагранжа для $n=2$ и $U=x_1 x_2$:
$$L(x_1, x_2, \lambda) = x_1 x_2 - \lambda(p_1 x_1 + p_2 x_2 - I) \rightarrow \max$$

Необходимое условие экстремума:
\begin{equation*}
	\begin{cases}
		\dfrac{\partial L}{\partial x_1} = x_2 - \lambda p_1 = 0\\[8pt]
		\dfrac{\partial L}{\partial x_2} = x_1 - \lambda p_2 = 0\\[8pt]
		\dfrac{\partial L}{\partial \lambda} =p_1 x_1 + p_2 x_2 - I = 0\\[8pt]
	\end{cases}
\end{equation*}

\begin{equation*}
	\begin{cases}
		x_2 = \lambda p_1\\
		x_1 = \lambda p_2\\
		p_1x_1+p_2x_2=I\\
	\end{cases}
\end{equation*}

\begin{equation*}
	\begin{cases}
		p_1 x_1 = p_2 x_2\\
		p_1x_1+p_2x_2=I\\
	\end{cases}
\end{equation*}

\begin{equation*}
	\begin{cases}
		x_1 = \dfrac{I}{2 p_1}\\[8pt]
		x_2 = \dfrac{I}{2 p_2}\\
	\end{cases}
\end{equation*}

Пусть $p_1=2,p_2=5,I=100$, тогда $x_1=25,x_2=10,U=250$.

Модель Стоуна.

$$U = \prod_{i=1}^{n} (x_i-a_i)^{b_i} \rightarrow \max$$
$$0 < b_i < 1$$

Где $a_i$ -- какой-то минимальный набор каждого блага, который должен включаться в потребительскую корзину.
Чем больше $b_i$, тем больше вклад блага $i$ в общую полезность.
$$\sum_{i=1}^{n} x_i p_i = I$$
$$\sum_{i=1}^{n} a_i p_i \le I$$

Решим оптимизиационную задачу потребительского выбора для модели Стоуна методом множителей Лагранжа:

$$L(x_1, \cdots, x_n, \lambda) = \prod_{i=1}^{n} (x_i-a_i)^{b_i} - 
\lambda \left(\sum_{i=1}^{n} x_i p_i - I\right) \rightarrow \max$$

\begin{equation*}
	\begin{cases}
		\dfrac{\partial L}{\partial x_1} = \dfrac{b_1}{x_1-a_1}\prod\limits_{i = 1}^{n} (x_i-a_i)^{b_i} - \lambda p_1 = 0\\[8pt]
		\cdots \\[8pt]
		\dfrac{\partial L}{\partial x_n} =\dfrac{b_n}{x_n-a_n}\prod\limits_{i = 1}^{n} (x_i-a_i)^{b_i} - \lambda p_n= 0\\[8pt]
		\dfrac{\partial L}{\partial \lambda} = \sum\limits_{i=1}^{n} x_i p_i - I = 0\\[8pt]
	\end{cases}
\end{equation*}


\begin{equation*}
	\begin{cases}
		\dfrac{b_k}{x_k-a_k}\prod\limits_{i = 1}^{n} (x_i-a_i)^{b_i} = \lambda p_k\\[8pt]
		\sum\limits_{i=1}^n  x_i p_i = I\\[8pt]
	\end{cases}
\end{equation*}

$$b_k \prod (x_i - a_i)^{b_i} = \lambda p_k (x_k-a_k)$$
Суммируем:
$$\sum_{k=1}^{n} b_k \prod (x_i - a_i)^{b_i} = \sum_{k=1}^{n}\lambda p_k (x_k-a_k)$$
$$\sum_{k=1}^{n} b_k \prod (x_i - a_i)^{b_i}  = \lambda \sum_{k=1}^{n} (p_k x_k-p_k a_k)$$
$$\sum_{k=1}^{n} b_k \prod (x_i - a_i)^{b_i}  = \lambda (I - \sum_{k=1}^{n} p_k a_k)$$

\begin{equation*}
	\begin{cases}
		b_k U = \lambda p_k (x_k - a_k)\\[8pt]
		\sum_{i=1}^{n} b_i I  = \lambda (I - \sum_{k=1}^{n} p_k a_k)\\[8pt]
	\end{cases}
\end{equation*}

\begin{equation*}
	\begin{cases}
		\lambda = \dfrac{\sum_{i=1}^{n}b_i U}{I - \sum_{i=1}^{n}a_ip_i}\\[8pt]
		b_k U = \dfrac{\sum_{i=1}^{n}b_i U}{I - \sum_{i=1}^{n}a_ip_i} p_k (x_k - a_k)\\[8pt]

$$ p_k (x_k - a_k) = \dfrac{b_k}{\sum_{i=1}^{n}b_i} \cdot \left(I - \sum_{i=1}^{n}a_ip_i\right)$$
